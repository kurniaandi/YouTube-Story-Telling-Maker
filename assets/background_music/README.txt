PLACEHOLDER - Background Music Files

Folder ini untuk menyimpan file musik latar royalty-free.

Format yang didukung:
- .mp3
- .wav
- .m4a
- .aac

Sumber musik royalty-free yang direkomendasikan:
1. Pixabay Music: https://pixabay.com/music/
2. YouTube Audio Library: https://studio.youtube.com/
3. Free Music Archive: https://freemusicarchive.org/
4. Incompetech: https://incompetech.com/music/royalty-free/

Instruksi:
1. Download musik instrumental yang sesuai tema motivasi/inspirasi
2. Pastikan musik adalah royalty-free atau Anda punya lisensi
3. Simpan file di folder ini
4. Pipeline akan otomatis pilih random dari file yang tersedia

Contoh nama file:
- inspirational_piano_01.mp3
- motivational_ambient_02.mp3
- uplifting_strings_03.mp3

PENTING:
- Jangan upload musik berhak cipta ke YouTube
- Cek lisensi setiap file sebelum digunakan
- Untuk testing, Anda bisa gunakan musik dari Pixabay (100% gratis)

File dummy untuk testing akan dibuat otomatis jika folder kosong.
